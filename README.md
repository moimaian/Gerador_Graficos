# Gerador_Graficos:
This is a simple Python interface focused on generating high-resolution graphs for scientific work.

## Installation:
### Files Download:
Download the GraphGenerator.py, GraphGenerator_Icon.png, and GraphGenerator_test.csv files to a folder within your personal Home folder. 
### Script execution
Open a terminal within this folder and run: 
'$python3 GraphGenerator.py'
or
'$python GraphGenerator.py'

Warning: In some cases, it is necessary to run the following beforehand: 
`$pip install pandas matplotlib`
